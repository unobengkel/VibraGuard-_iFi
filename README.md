# VibraGuard-_iFi
VibraGuard WiFi adalah solusi keamanan mandiri (DIY) yang menggunakan sensor getaran SW-420 dan Wemos ESP8266 untuk mendeteksi upaya paksa pada pintu. Alat ini dilengkapi dengan Web Dashboard real-time yang dapat diakses melalui WiFi lokal tanpa memerlukan koneksi internet, sehingga privasi data lebih terjaga.
